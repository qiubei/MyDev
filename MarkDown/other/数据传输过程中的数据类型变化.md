# 数据传输过程中的数据类型变化

1. `硬件 -> app`

    本地缓存的是 byte 的序列（[UInt8]）。

2. `app -> 服务器`

    使用 json 格式，但 json 不支持 UInt8，传入服务器前 app 将数据转成 Int。服务器拿到的是整型序列（[Int]）。
    
3. `服务器数据处理`

    服务器使用数据要记住这个序列是 Int 而不是 UInt8.（例如：如果要把原始文件格式和以前的格式统一，需要将 Int 转成 UInt8 之后再写入文件。）
    
4. `服务器 -> 算法`

    你需要确认算法的输入是什么以及它的数据类型，确定好你反馈给子豪。
